## 6-1: 기본 기능 테스트 (15-20분)

### 테스트 절차

#### 1단계: ROM 로드 (1분)
```
1. VisualBoyAdvance M 실행
2. File > Open ROM
3. output/game_wars_korean_final.gba 선택
4. 로드 확인 (오류 메시지 없음)
5. 게임 화면 표시 확인
```

**예상 결과**: 게임 타이틀 화면 표시 (Game Wars 1+2)

**실패 시**: 
- 에뮬레이터 오류 메시지 기록
- ROM 파일 손상 가능성 → tools/phase6_basic_test.py 재실행

#### 2단계: 초기 화면 네비게이션 (5분)
```
메인 메뉴 → 게임 선택
[ ] Game Wars 1 선택 가능
[ ] Game Wars 2 선택 가능
[ ] 게임 시작 가능

뒤로가기 가능 확인
```

**체크 포인트**:
- 한글 메뉴 텍스트 렌더링 확인
- 텍스트 오버플로우 없음
- 메뉴 네비게이션 정상 작동

#### 3단계: 게임 시작 (10분)

**Game Wars 1**:
```
게임 선택 → 난이도 선택 → 스토리 모드 시작

[ ] 난이도 선택 화면 표시
[ ] 스토리 모드 진입 가능
[ ] 첫 번째 미션 맵 로드
[ ] 유닛 배치 화면 표시
```

**관찰 사항**:
- [ ] 한글 UI 텍스트 렌더링
- [ ] 유닛명 (한글) 표시
- [ ] 아이템 설명 (한글) 표시
- [ ] 상태창 텍스트 정상 표시

#### 4단계: 게임플레이 기본 (5-10분)

```
첫 미션에서:
[ ] 유닛 선택 가능

[ ] 전투 시작 가능
```

## 6-2: 텍스트 렌더링 상세 검증 (20-30분)

### 한글 텍스트 렌더링 확인

#### 메뉴 텍스트 (5분)
```
[ ] 메인 메뉴: 한글 렌더링 확인
    - Game Wars 1
    - Game Wars 2
    - 설정
    - 종료

[ ] 게임 선택: 한글 렌더링
    - 난이도 선택
    - 스토리 모드
    - 기타 모드

[ ] 설정 메뉴: 한글 렌더링
    - 음량 조절
    - 속도 설정
    - 저장/로드
```

**기록 항목**:
- [ ] 가독성 평가 (1-5: 5=완벽)
- [ ] 폰트 크기 (너무 크거나 작음?)
- [ ] 글자 깨짐 또는 파손 여부

#### UI 텍스트 (10분)

게임 진행 중 다음 UI 요소 확인:

```
상태창:
[ ] HP/MP 숫자 정상 표시
[ ] 속도, 방어력 등 스탯 정상 표시
[ ] 상태 효과 텍스트 한글 표시

대사 상자:
[ ] 캐릭터 이름 한글 표시
[ ] 대사 텍스트 한글 표시
[ ] 텍스트 정렬 정상

메시지 상자:
[ ] "미션 완료" 메시지
[ ] "경험치 획득" 메시지
[ ] "아이템 획득" 메시지
```

#### 텍스트 Overflow 확인 (10분)

```
확인할 항목:
[ ] 메뉴 텍스트 잘림
    - 해당 시 길이: ___ 자
    - 할당 공간: ___ 픽셀
    
[ ] 대사 상자 overflow
    - 텍스트 넘침 여부
    - 줄바꿈 정상 작동
    
[ ] 유닛/아이템 이름 잘림
    - 긴 이름 표시 상태
    - 공간 부족 표시
```

**텍스트 Overflow 기록 템플릿**:
```
주소: 0x??????
원본: [일본어 텍스트]
번역: [한글 텍스트]
길이: 원본 N바이트 → 번역 M바이트
문제: 오버플로우/정상
스크린샷: (필요시)
```

## 6-4: 버그 분류 및 기록

### 버그 심각도 분류

**Critical** (게임 진행 불가)
- 게임 크래시
- 시스템 오류로 진행 불가
- 저장/로드 불가

**Major** (기능 오류)
- 텍스트 완전 깨짐
- 메뉴 네비게이션 오류
- 전투 시스템 오류

**Minor** (표시 오류)
- 한글 텍스트 부분 깨짐
- 오타 또는 문법 오류
- 폰트 렌더링 이상 (가독성 상 문제)

**Enhancement** (개선사항)
- 번역 개선 제안
- UI 개선 제안
- 성능 개선
