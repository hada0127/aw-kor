# 무라마사 한글 재번역 말투·표기 가이드

대사 재번역(이슈 #9) 작업 시 반드시 따른다. gemini 배치 번역 프롬프트 + claude 검수 기준.

## 전체 톤
- 에도~전국시대 **사극풍**. 과한 현대어·신조어 지양하되 고어로 난해하지 않게(가독성 우선).
- 한 캐릭터 내 종결어미 일관(말투 급변 금지).
- **원문 정보 누락 복원**: 폰트 부족 시절 2줄→1줄로 버린 정보를 되살린다.

## 표기·형식 규칙
- **인명/지명**: `proper_nouns.json` 표준 표기 유지. (독법 확정본 따름 — 위키 재확인 중)
- **시대 용어**: 현대어 금지. 예) 이장→**명주(名主)**, (농사 강조 시) 백성→**농사꾼**(百姓).
- **폭 29.5전각/줄**(잘림 한도, 원작 ja max31 기준), 미관 여백 27, **최대 3줄**(4줄 금지).
- **마침표** + 줄임표(…)·마침표(.) 뒤 같은 줄에 문장 이어지면 **공백 1칸**. 문장 시작 줄임표(…말)는 공백 없이.
- 일본어 문장부호(。、「」)·가나·한자 잔존 0. 매핑 외 글자(완성형 2350 밖) 0.

## 작업 파이프라인
1. `score_candidates.py` 우선순위 → 2. gemini 배치 번역(이 가이드+사전+ja+현 ko)
→ 3. claude 검수(`lint_dialogs.py` + 폭/인명 수동) → 4. 통과분만 적용
→ 5. 전체 후 gemini 최종 검증 → 6. 빌드.
