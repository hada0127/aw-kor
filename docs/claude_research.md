# GBA 한글화 프로젝트 - Claude 검색 결과

## 개요
Game Boy Advance(GBA) 게임의 한글화(로컬라이제이션)에 대한 커뮤니티 기반 정보와 도구들을 정리한 문서입니다.

## 1. 한글 패치 커뮤니티 및 리소스

### 주요 웹사이트
- **한글로게임**: GBA 한국어 패치 리스트를 제공하는 커뮤니티 사이트 (https://www.hangulogame.com/lists/gba/)
- **GitHub - tentacleteam/romlistkr**: 텐타클 팀의 레트로게임 롬 리스트 프로젝트 (https://github.com/tentacleteam/romlistkr)
- **나무위키**: 게임보이 어드밴스 정보 (https://namu.wiki/w/게임보이%20어드밴스/)

### 한글 패치 작업 커뮤니티
- 네이버 카페에서 한글화하는 사람들이 모여 활동 중
- 포켓몬, 메트로이드, Fire Emblem 등 다양한 게임의 한글화 진행 중

## 2. 에뮬레이터

### 주요 에뮬레이터
- **VisualBoyAdvance M v2.1.4.x64**: Windows용 GBA 에뮬레이터
  - 마우스 끌어다 놓기(Drag & Drop)로 실행 가능
  - 디버그 기능으로 메모리 접근 및 스프라이트 추출 가능

### 에뮬레이터의 활용
- 메모리 디버깅을 통한 게임 분석
- Hex 코드 및 팔레트 코드 읽기
- 스프라이트 추출

## 3. 한글 패치 작업 프로세스

### 포켓몬 에메랄드 예시
실제 한글화 작업은 다음 단계로 진행됩니다:
1. 소스 코드 준비
2. 스크립트 번역
3. 소스 코드 내 번역 삽입
4. 배틀 대사 출력 조정
5. 도감 위치 조정
6. 자동화 빌드 프로세스

### 작업 규모
- 수십~100개 이상의 커밋으로 나뉨
- 각 게임의 구조에 따라 달라짐

## 4. 자동화 도구 및 배포

### Pack.bat 기반 자동화
- 번역 후 Pack.bat 명령어로 자동화
- 폰트 생성 및 삽입 자동화
- 대사 입력 자동화

### 도구 활용
- tadpole 프로그램: ROM 변환 및 수정의 핵심 도구
- 각 게임별로 커스터마이즈된 빌드 스크립트 필요

## 5. 포켓몬 게임 한글화 예시

### 이용 가능한 한글 롬
- 1세대~3세대 포켓몬 게임의 한글 롬 존재
- 커뮤니티 사이트에서 다운로드 가능
- 기본 롬(영문/일본어)에 한글 패치 파일 적용

## 6. Game Wars 한글화 관련
현재 Game Wars의 구체적인 한글화 정보는 온라인에서 많이 찾기 어려움.
다음 작업이 필요:
- 게임 스크립트 분석
- 텍스트 추출 및 테이블 작성
- 커뮤니티 접촉

## 참고 사항
- 대부분의 한글화는 개인 개발자나 소규모 커뮤니티에서 진행
- 각 게임마다 구조가 다르므로 게임별 맞춤형 접근 필요
- ROM 해킹 커뮤니티의 지원과 공유 문화가 중요
