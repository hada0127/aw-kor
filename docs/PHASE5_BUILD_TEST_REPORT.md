## 기술적 발견사항

### 1. ROM 인코딩 체계
- **일본어**: Shift-JIS 2바이트 인코딩
- **문자 맵**: 3,568개 고유 바이트-문자 쌍
- **ASCII**: 0x40-0x7E 범위 표준 ASCII

### 2. 포인터 관리
- **포맷**: Little-endian 4바이트 또는 2바이트
- **검증**: 인코딩/디코딩 일치 확인됨
- **범위**: 전체 16MB 주소 공간 지원

### 3. ROM 체크섬
- **알고리즘**: GBA 표준 체크섬
- **위치**: 0xBD 바이트
- **계산**: 0xA0-0xBD 범위 합계 + 0x19
- **결과**: 원본 ROM 검증됨

### 4. 파일 시스템
- **구조**: 표준 프로젝트 디렉토리 구조
- **용량**: 출력 디렉토리 충분 (16MB ROM 저장 가능)
- **권한**: 읽기/쓰기 정상
