## 기술 난제 분석

### 공간 부족 (107개 실패)
**문제**: 일부 한글 번역이 원본 일본어보다 길어서 할당 공간 초과

**현황**:
- 총 바이트 계산: 한글이 120 bytes 더 효율적
- 개별 항목: 107개는 공간 부족, 114개는 여유

**해결 방안**:
1. 텍스트 단축 (한글 표현 최적화)
2. 공간 재배치 (복잡함, 포인터 업데이트 필요)
3. 폰트 크기 축소 (기술적 난제)

### 폰트 위치 미확인
**현황**: 0x400000 추정값 (검증 필요)

**영향**: 
- 한글 폰트 교체 예정시 정확한 위치 필요
- 현재는 기존 일본어 폰트 사용

**해결**: HxD로 수동 확인 (도구 제약)
